Get battery stats

Let's get started! Did you know that you can get battery level of user's device? To do it you have to call navigator.getBattery function. It returns a battery promise. The battery level is being stored in "level". iOS (iphones and ipads) does not support battery level detection with JS.


navigator.getBattery().then(
function(battery){
alert("Your battery level is: " + battery.level * 100 + "%"); })

Vibration

With the navigator.vibrate(intvalue) you can vibrate user's device. This function accepts one integer value - the vibration duration in milliseconds.


navigator.vibrate(1000);

You can also vibrate with a pattern. To do it you have to use the navigator.vibrate() function again, but with an array passed in now. The first item of the array (50) is the duration of vibration, second (100) is a break, third is vibration again and so on.


navigator.vibrate([50, 100, 50, 100, 200, 10, 10, 20, 20, 20, 20, 100, 100, 400, 50, 100, 50, 100, 200, 10, 10, 20, 20, 20, 20, 100, 100]);

Note: this will only work on a mobile device excluding iOS ones.

Is device offline or online?

To check if device is online you can use the onLine method of navigator object.


alert(navigator.onLine);

Copy stuff to clipboard

We can copy something to clipboard with document.execCommand('copy'); but it won't work if no text is selected. Let's create a text field and a button which will copy everything that user had entered there. Now we can use document.getElementById("input").select(); document.getElementById("input").setSelectionRange(0, 99999); to select text inside our text field. Let's put all our code above in a function and assign is as an onclick property of a button. Putting everything together:


function copySomething(){ document.getElementById("input").select(); document.getElementById("input").setSelectionRange(0, 99999); document.execCommand('copy'); alert("Copied! Now try to paste the text you just copied into the text field below") }

Try it out!



Get OS info

Use navigator.userAgent to get OS info.


alert(navigator.userAgent);
